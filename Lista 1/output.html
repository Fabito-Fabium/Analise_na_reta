1. Como vimos, existe uma função injetora s : ℕ → ℕ, tal que s(n) = n + 1, em que s(n) chama-se sucessor de n. A partir disso, podemos definir a operação soma em ℕ, que satisfaz a seguinte lei de recursão:

    • Para todo n ∈ ℕ, temos que n + 1 = s(n);

    • Para cada m, n ∈ ℕ, temos que n + s(m) = s(n+m).

    Também podemos definir a operação de produto em ℕ, que satisfaz a seguinte lei de recursão:

    • Para todo n ∈ ℕ, temos que n ⋅ 1 = n;

    • Para cada m, n ∈ ℕ, temos que n ⋅ s(m) = nm + n.

    Mostre que valem as seguintes propriedades:

    1. (Comutatividade da soma) Para cada m, n ∈ ℕ, temos que m + n = n + m.

    2. (Lei do cancelamento da soma) Para cada m, n, p ∈ ℕ, se m + p = n + p, então m = n.

    3. (Comutatividade do produto) Para cada m, n ∈ ℕ, temos que mn = nm.

    4. (Distributividade) Para cada m, n, p ∈ ℕ, temos que m(n+p) = mn + mp.

    5. (Lei do cancelamento do produto) Para cada m, n, p ∈ ℕ, se mp = np, então m=n.

    6. (Unicidade da identidade do produto) Para todo n ∈ ℕ, se mn = n, então m = 1

  2. Sejam m, n, p ∈ ℕ. Mostre que

    1. Se m ≤ n e n < p, então m < p.

    2. Se m < n e n ≤ p, então m < p.

    3. Se m ≤ n e n ≤ p, então m ≤ p.
